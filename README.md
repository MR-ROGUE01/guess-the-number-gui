# Guess The Number (Python GUI)

A beginner-level number guessing game built using Python and Tkinter.

## Overview
This project is a GUI-based game where the computer guesses a number within a user-defined range using logical narrowing. 
It was created to practice event-driven programming and basic state management in Python.

## Tech Used
- Python
- Tkinter (GUI)

## Purpose
Built as part of my learning journey to understand GUI development, event handling, and game logic in Python.
